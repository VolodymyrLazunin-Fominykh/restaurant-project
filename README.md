# La Bella Cucina - Restaurant Website

A professional 4-page restaurant website built in Week 2 of my coding bootcamp.

## 🌐 Live Demo
[View Live Site →](https://novea-restaurant-project.netlify.app)

## 📄 Pages
- **Home** - Hero, features, highlights
- **Menu** - Full menu with categories and pricing
- **Our Story** - About page with testimonials
- **Contact** - Reservation form and info

## 🛠️ Technologies
- HTML5 (multi-page)
- CSS3 (CSS Variables, Flexbox, Grid)
- CSS Animations & Keyframes
- Google Fonts (Playfair Display, Lato)
- Responsive Design

## 🎯 What I Learned
- CSS custom properties (variables) for theming
- CSS animations with @keyframes
- Multi-page site structure and navigation
- Advanced typography with Google Fonts
- CSS ::before / ::after pseudo-elements
- object-fit for images
